# ai-investments-3d
Интерактивное веб-приложение для поиска оптимальных стратегий с 3D-визуализацией пространства решений.
Интерактивное веб-приложение для поиска оптимальных стратегий с 3D-визуализацией пространства решений.

## Возможности
- Генерация тысяч сценариев методом случайного поиска
- Интерактивная 3D-диаграмма (Plotly) с вращением и зумом
- Цветовая кодировка: зеленые точки близки к цели, красные далеки
- Выделение топ-3 лучших стратегий

## Запуск локально
```bash
pip install -r requirements.txt
streamlit run app.py
